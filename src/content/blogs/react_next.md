#### server side rendering(SSR) and client side rendering(CSR)

feature of next.js, by 'use client' at the top of a file

SSR: the content of website will be rendered on server side and send the processed html files to client side, which can users have a faster experience.

CSR: the content of website will not pre-render on server side,  instead, an html file with some js files will be sent to client side and rendered by browser, thus users can feel a little slow.
